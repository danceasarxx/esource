# esource

A simple implementation of the basic ideas of event sourcing.
It is made up of three parts:

+ A store
+ A bus
+ A stream

## Installation
Add the following dependency to your `project.clj` file:

    [esource "0.1.0"]

## Usage

```clojure
(use 'esource.core)
```
